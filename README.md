# Food Hub

**Food Hub** is a modern food ordering web application built with React. It features client-side routing with React Router, state management with Redux Toolkit, and styling using Tailwind CSS. The project uses Parcel as the bundler and Jest for unit testing.

---

## Demo

[Live Demo on Netlify](https://foodhub-app.netlify.app)

---

## Features

- Responsive React UI with multiple pages (Home, About, Contact, Grocery, Restaurant Menu, Cart)
- React Router v6 for declarative client-side routing
- Lazy loading of components using React `Suspense` and `lazy`
- Global state management with Redux Toolkit and React-Redux
- Context API usage for user state management
- Styled with Tailwind CSS v4 for utility-first styling
- Unit testing with Jest and React Testing Library
- Bundled with Parcel for zero-config build and fast development

---

## Tech Stack

- React 19
- React Router DOM 6.21
- Redux Toolkit & React Redux
- Tailwind CSS v4
- Parcel Bundler
- Jest & React Testing Library
- Axios (for API calls)
